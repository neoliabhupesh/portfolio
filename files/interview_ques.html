Explain what is the difference between "visibility:hidden" and "display:none"

They are both style properties
visibility:hidden: This property hides the element, but it still takes up space in the layout
display:none: It eliminates the element completely from the document. It does not take up any space, even though the HTML for it is still in the source code.


How you use float in CSS?

In CSS, we would use CSS float when you want to make an element of your page be pushed to the right or left and make other elements wrap around it. but it is necessory to clear the float after shifting the elements to the respective direction as it is the sister property of float, otherwise we will get the next element adjacent to the float element. By clearing the float we will find the next element down to the past float element.
#footer { clear: both; }


Explain how can you use one CSS file to multiple webpage externally?

To use one CSS file with multiple webpage , we have to link the file inside the <head> tag using "Link" property.
<link rel="stylesheet" type="text/css" href="css/style.css">

Explain viewport and it attributes?

View Port is the area which covers the user's screen and the visible part.
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<meta> controls/provides the basic browser instruction, these tags are placed inside head section of html code used by the search engine to display description of page in search results.
